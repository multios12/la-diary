# la-diary
  this app is line diary.

## デバッグ実行
vscode上での実行を前提。chromeを利用
フロントデバッグサーバ：ポート
バックエンドサーバ：ポート80

### 実行手順
1. 実行とデバッグで「go API Server」を選択、実行
2. 実行とデバッグで「debug react」を選択、実行
